# 💙 Heart Disease Prediction Web App (AI Project)

This is a Machine Learning project built using Python and Streamlit.  
The system predicts the risk of heart disease based on patient health parameters.

---

## 🚀 Features

- Machine Learning model using Logistic Regression
- Interactive user-friendly web interface
- Real-time prediction system
- Clean and peaceful UI design
- Easy input form for medical attributes

---

## 🧠 Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

---

## 📊 How It Works

1. User enters patient health details
2. Data is processed by trained ML model
3. Model predicts whether the patient is at risk
4. Result is shown instantly on the UI

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
